SPIDER SENSE

It is a browser extension which helps users monitor their Scrapy Cloud spider jobs. It’s essentially a dashboard. It is generally used by the people who create web crawlers which is also called spiders.

Role

UI/UX Designer

Team

Solo

Duration

2 weeks

PROBLEM

How to help the existing product to make it easy to use?

I found an issue on Github which was posted by a friend and the description was there to change the design to make it more appealing.

I wanted to figure out how can I help the current product to maximize its usage.

SOLUTION

Redesigned the existing UI of the product.

I did some competition analysis and user research by conducting interview for the existing product and found out the problems which were being faced by the users.

I found out that the main problem was UI so I worked mostly on the UI keeping the users in the mind at all times.

RESEARCH

I conducted 4 user interviews.

In order to understand the problem being suffered by the users I conducted user interview.

The major reason portrayed by the user was that the look and feel of the product was not catchy. They were finding it difficult to use the product as there was too much information on the page.

IDEATION

The next step was to generate ideas.

To generate ideas we (I and the existing users) went for the storyboarding method. In this way we could trace down all the situations.

We found out that the dashboard should be -

  1. Clean
  2. Accurate
  3. Informative
  4. User - friendly
